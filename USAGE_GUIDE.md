# คู่มือการใช้งานระบบ Library ที่ปรับปรุงแล้ว

## ภาพรวมการปรับปรุง

โปรเจกต์นี้ได้รับการปรับปรุงหน้า Library ให้มีฟีเจอร์ครบถ้วนเหมือนกับหน้าข่าวสาร พร้อมระบบ Filter และการลิงค์ไปยังหน้ารายละเอียด

---

## ฟีเจอร์ที่เพิ่มเข้ามา

### 1. หน้า Library แบบใหม่ (`/library`)

**ฟีเจอร์:**
- แสดงสถานที่ท่องเที่ยวทั้งหมด 19 แห่ง ในรูปแบบ Grid
- ระบบ Filter แบ่งตามเขต: พัทยาเหนือ, พัทยากลาง, พัทยาใต้, นาเกลือ, จอมเทียน, เกาะล้าน
- แต่ละการ์ดแสดง:
  - รูปภาพสถานที่
  - ชื่อสถานที่
  - หมวดหมู่ (แสดงเป็น badge บนรูป)
  - คำอธิบายย่อ
  - จำนวนครั้งที่เข้าชม
- คลิกที่การ์ดเพื่อดูรายละเอียดเพิ่มเติม

**วิธีใช้งาน:**
1. เปิดหน้า Library จาก Navbar หรือไปที่ `/library`
2. คลิกปุ่ม "ทั้งหมด" เพื่อดูสถานที่ท่องเที่ยวทั้งหมด
3. คลิกปุ่มหมวดหมู่ (เช่น "พัทยาเหนือ") เพื่อกรองเฉพาะเขตนั้นๆ
4. คลิกที่การ์ดสถานที่ท่องเที่ยวเพื่อดูรายละเอียด

---

### 2. หน้ารายละเอียดสถานที่ท่องเที่ยว (`/library/:id`)

**ฟีเจอร์:**
- แสดงรูปภาพหน้าปกขนาดใหญ่
- แสดงชื่อสถานที่และหมวดหมู่
- แสดงจำนวนครั้งที่เข้าชม
- แสดงรายละเอียดสถานที่
- แสดงที่อยู่พร้อมไอคอน
- แสดงเวลาเปิด-ปิดพร้อมไอคอน
- มีปุ่ม "กลับ" เพื่อกลับไปหน้า Library
- มีส่วนแสดงความคิดเห็น (Comment Section)

**วิธีใช้งาน:**
1. คลิกที่การ์ดสถานที่ท่องเที่ยวจากหน้า Library
2. ดูรายละเอียดครบถ้วนของสถานที่
3. คลิกปุ่ม "กลับ" เพื่อกลับไปหน้า Library

---

### 3. Footer ที่ปรับปรุงแล้ว

**ฟีเจอร์:**
- ลิงค์ใน "Pattaya Library" นำไปยังหน้า `/library`
- ลิงค์ใน "หมวดหมู่" นำไปยังหน้า `/news`
- ลิงค์ใน "เกี่ยวกับเรา" นำไปยังหน้า `/` (Home)
- ใช้ React Router Link สำหรับการนำทางที่รวดเร็ว (ไม่ reload หน้า)

**วิธีใช้งาน:**
1. เลื่อนลงมาที่ Footer ของหน้าใดก็ได้
2. คลิกลิงค์ที่ต้องการเพื่อนำทางไปยังหน้านั้นๆ

## การทดสอบ

### ทดสอบ Filter:
1. ไปที่หน้า `/library`
2. คลิก "พัทยาเหนือ" → ควรแสดง 4 สถานที่
3. คลิก "พัทยากลาง" → ควรแสดง 4 สถานที่
4. คลิก "ทั้งหมด" → ควรแสดง 19 สถานที่

### ทดสอบการลิงค์:
1. คลิกที่การ์ดสถานที่ใดก็ได้ → ควรไปยังหน้ารายละเอียด
2. คลิกปุ่ม "กลับ" → ควรกลับไปหน้า Library
3. คลิกลิงค์ใน Footer → ควรนำทางไปยังหน้าที่ถูกต้อง

### ทดสอบ Responsive:
1. ลองเปิดดูในหน้าจอขนาดต่างๆ (Desktop, Tablet, Mobile)
2. ตรวจสอบว่า Grid ปรับขนาดได้ถูกต้อง
3. ตรวจสอบว่าปุ่ม Filter แสดงผลดีในทุกขนาดหน้าจอ

## การแก้ไขข้อมูลสถานที่

หากต้องการแก้ไขหรือเพิ่มสถานที่ท่องเที่ยว:

1. เปิดไฟล์ `frontend/src/data/libraryData.js`
2. แก้ไขหรือเพิ่มข้อมูลในอาร์เรย์ `allLibraries`
3. รูปแบบข้อมูล:
```javascript
{
  id: 20,                           // ID ที่ไม่ซ้ำกัน
  title: 'ชื่อสถานที่',
  location: 'พัทยาเหนือ',           // หมวดหมู่
  views: 0,                         // จำนวนครั้งที่เข้าชม
  img: 'URL รูปภาพ',
  description: 'คำอธิบาย',
  address: 'ที่อยู่',
  openingHours: 'เวลาเปิด-ปิด',
}
```

## ปัญหาที่อาจพบและวิธีแก้ไข

### ปัญหา: Filter ไม่ทำงาน
**วิธีแก้:** ตรวจสอบว่า `location` ในข้อมูลตรงกับชื่อหมวดหมู่ที่ต้องการ

### ปัญหา: การ์ดไม่แสดงรูปภาพ
**วิธีแก้:** ตรวจสอบ URL รูปภาพใน `libraryData.js` ว่าใช้งานได้

### ปัญหา: ลิงค์ใน Footer ไม่ทำงาน
**วิธีแก้:** ตรวจสอบว่าได้ import `Link` จาก `react-router-dom` แล้ว

### ปัญหา: หน้ารายละเอียดไม่แสดง
**วิธีแก้:** ตรวจสอบว่าได้เพิ่ม route `/library/:id` ใน `App.jsx` แล้ว
