# สรุปการพัฒนาเว็บไซต์ข่าว - Customer Project

## ภาพรวมโปรเจกต์
เว็บไซต์ข่าวที่พัฒนาด้วย React + Vite โดยมีการจัดการข่าวทั้งหมด 10 ข่าว แต่ละข่าวมี ID เฉพาะตัว (1-10)

## สิ่งที่ได้ทำเสร็จ

### 1. สร้างระบบข้อมูลข่าวแบบรวมศูนย์
- ไฟล์: `/src/data/newsData.js`
- ข่าวทั้งหมด 10 ข่าว พร้อม ID ที่ไม่ซ้ำกัน (1-10)
- แบ่งข่าวออกเป็นกลุ่ม:
  - **Grid News** (ID: 1-4): ข่าวที่แสดงในส่วน NewsGrid
  - **Popular News** (ID: 5-8): ข่าวยอดนิยม
  - **Slider News** (ID: 9-10): ข่าวที่แสดงใน Slider

### 2. สร้างหน้ารายละเอียดข่าว (NewsDetail)
- ไฟล์: `/src/pages/NewsDetail.jsx`
- ไฟล์ CSS: `/src/css/NewsDetail.css`
- ฟีเจอร์:
  - แสดงรายละเอียดข่าวแบบเต็ม
  - แสดง ID ของข่าว
  - Breadcrumb navigation
  - แสดงหมวดหมู่, วันที่, จำนวนผู้เข้าชม
  - แสดงข่าวที่เกี่ยวข้อง (ข่าวในหมวดเดียวกัน)
  - ใช้เทมเพลตเดียวกับหน้าแรก (Navbar + Footer)

### 3. อัปเดต Routing
- ไฟล์: `/src/App.jsx`
- เพิ่ม route: `/news/:id` สำหรับหน้ารายละเอียดข่าว

### 4. อัปเดต Components ให้เชื่อมโยงกับข้อมูลกลาง

#### NewsGrid Component
- ไฟล์: `/src/components/NewsGrid.jsx`
- ดึงข้อมูลจาก `getGridNews()`
- เพิ่มลิงก์ไปยังหน้ารายละเอียดข่าว
- แสดงข่าว ID: 1, 2, 3, 4

#### Popularcard Component
- ไฟล์: `/src/components/Popularcard.jsx`
- ดึงข้อมูลจาก `getPopularNews()`
- เพิ่มลิงก์ไปยังหน้ารายละเอียดข่าว
- แสดงข่าว ID: 5, 6, 7, 8

#### ImageSlider Component
- ไฟล์: `/src/components/ImageSlider.jsx`
- ดึงข้อมูลจาก `getSliderNews()`
- เพิ่มลิงก์ไปยังหน้ารายละเอียดข่าว
- แสดงข่าว ID: 9, 10

### 5. อัปเดต CSS
- อัปเดต `NewsGrid.css` เพื่อรองรับ Link
- อัปเดต `Popularcard.css` เพื่อรองรับ Link
- อัปเดต `ImageSlider.css` เพื่อรองรับ Link
- สร้าง `NewsDetail.css` ใหม่สำหรับหน้ารายละเอียด

### 6. ปรับแต่ง Vite Configuration
- ไฟล์: `vite.config.js`
- เพิ่ม server configuration:
  - `host: '0.0.0.0'` - อนุญาตการเข้าถึงจากภายนอก
  - `port: 5173` - กำหนด port
  - `allowedHosts` - อนุญาต domain ที่ใช้ทดสอบ

## วิธีการรันโปรเจกต์

```bash
cd /home/ubuntu/Customer-project/frontend
pnpm install
pnpm run dev
```

เว็บไซต์จะรันที่: http://localhost:5173

## URL สำหรับทดสอบ

- หน้าแรก: `/`
